# Setup Instructions

1. Clone the repository:

   ```bash
   git clone <repository-url>
   cd ravenpack-userflag-service-exercise
   ```

2. Build the project:

   ```bash
   ./mvnw clean install
   ```

3. Run the application:

   ```bash
   ./mvnw spring-boot:run
   ```

4. Access the application at `http://localhost:8080`.
